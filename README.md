# Taekwondo Scoreboard (Client)

Tech: React, TailwindCSS, SocketIO

### Installation
1. Clone this repository
2. Run 
```bash
npm install 
```
or yarn 
```bash
yarn install 
```
### Set up environment

Provide port, socket host in ``.env`` (create if not exists)

```
PORT=1234
```

### Run application
Dev mode:
```bash
yarn run dev
```

Build & run in product mode:
```bash
yarn run build
yarn start
```
